# Text to Speech (TTS) Recorder 🎙️📖

A versatile and high-performance **Text-to-Speech (TTS) Utility** built with **Flutter**. This application allows users to instantly transform text into natural-sounding speech, customize playback parameters, and export the audio as high-quality **WAV files**.

## 🌟 Key Features
- **Extensive Language Support:** Access over 300 voices and languages (depending on device capabilities).
- **Record & Export:** Capture speech and save it directly as a high-quality **WAV file** for use in other projects.
- **Dynamic Customization:** Fine-tune the listening experience by adjusting **speed** and **pitch** in real-time.
- **Smart Presets:** Save up to **9 text presets** for instant switching and recurring use.
- **Easy Sharing:** Seamlessly share your generated audio files with other apps and platforms.

## 💡 Best For...
- **Content Creators:** Generating narrations for videos or presentations without expensive equipment.
- **Language Learners:** Practicing pronunciation and listening to text in various dialects.
- **Accessibility:** Assisting those who prefer auditory learning or need text-to-voice support.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Integration with native Text-to-Speech engines and audio buffer management for high-fidelity WAV encoding.

*Note: This repository contains the core logic and source code. Assets (images, fonts, and localization files) are excluded to protect intellectual property.*
